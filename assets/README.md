# Portfolio Assets

This folder contains all the images and assets used in the portfolio website.

## Required Images

### Hero Background
- **File**: `hero-bg.jpg`
- **Dimensions**: 1920x1080 (or larger)
- **Description**: A professional, high-quality background image for the hero section. Should be suitable for overlay text.
- **Suggestion**: Use a subtle, professional image related to data analytics, technology, or abstract patterns.

### Profile Image
- **File**: `profile.jpg`
- **Dimensions**: 400x400 (square)
- **Description**: Professional headshot of Basit Ali
- **Format**: JPG or PNG with good quality

### Project Images
- **File**: `project1.png` - Traffic Accidents Analysis project
- **File**: `project2.png` - Azure Data Pipeline Optimization project  
- **File**: `project3.png` - NLP Business Data Insights project
- **Dimensions**: 600x400 (3:2 aspect ratio)
- **Description**: Screenshots or mockups of the projects
- **Format**: PNG or JPG

## Image Guidelines

1. **Optimization**: All images should be optimized for web use to ensure fast loading times
2. **Quality**: Use high-quality images that reflect professionalism
3. **Consistency**: Maintain consistent style and quality across all images
4. **Alt Text**: Remember to update alt text in HTML if images change

## Placeholder Images

If you need placeholder images for development, you can use:
- [Unsplash](https://unsplash.com) for high-quality stock photos
- [Pexels](https://pexels.com) for free stock photos
- [Placeholder.com](https://placeholder.com) for quick placeholders
- [Lorem Picsum](https://picsum.photos) for random placeholder images

## File Naming Convention

- Use lowercase letters
- Use hyphens for spaces
- Be descriptive but concise
- Maintain consistent naming patterns
